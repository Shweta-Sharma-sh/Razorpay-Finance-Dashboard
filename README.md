# 💳 Razorpay Finance Dashboard

Welcome! 👋
This project is a simple finance dashboard inspired by Razorpay. It is built to explore how digital payments and transaction systems work using real-time data from public APIs.

---

## 💡 What is this project?

This is a web application that displays and manages transaction data.
It allows users to search, filter, and sort financial information in an interactive way.

The goal is to learn:

* API integration using JavaScript
* Working with real-world data
* Building responsive and clean UI

---

## 🎯 Vision

To create a simple and interactive platform that helps understand how modern payment systems like Razorpay function, while improving development skills in JavaScript and UI design.

---

## 🚀 Features

* 🔍 Search transactions
* 🎯 Filter data
* 🔃 Sort results
* 📱 Responsive design

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
